## Analysis

This directory contains security analysis work focused on identifying detection gaps, mapping attacker behavior, and understanding their impact on real-world SOC environments.

Some analyses are derived from operational scenarios and have been sanitized to remove sensitive details while preserving the underlying logic and findings.

### Case Studies

#### 1. MITRE ATT&CK Mapping Study
* Example attacker activity or command
* Relevant telemetry sources within Microsoft Sentinel
* Detection logic and example KQL queries

#### 2. Detection Gap Analysis

* Evaluates existing detections against attacker behavior
* Identifies missing coverage and reliance on specific log sources

#### 3. Threat Intelligence Report

* Analyzes selected threats and attacker techniques
* Explores how intelligence can be translated into detection opportunities
